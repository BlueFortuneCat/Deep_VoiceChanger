# Voice_DiscoGAN
キズナアイとねこますの声を入れ替える機械学習  
A DiscoGAN base voice changer.

# Usage
-v -w のオプションそれぞれにwaveファイルのパスを入れてください。  
Give path of wave file by -v and -w option.
